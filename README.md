# Sistema Bancário em Java (Console)

Projeto desenvolvido em Java com o objetivo de simular operações básicas de uma conta bancária, praticando lógica de programação, entrada e saída de dados e manipulação de variáveis.

---

## Funcionalidades

* Criação de conta bancária
* Registro do nome do titular
* Definição do número da conta
* Depósito de valor inicial
* Realização de saque
* Exibição de extrato com saldo final

---

## Tecnologias utilizadas

* Java
* Eclipse IDE

---

## Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/VictorSSouza013/sistema-bancario-java.git
```

2. Abra o projeto no Eclipse
3. Execute a classe `Main.java`
4. Insira os dados solicitados no console
